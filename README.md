
# O.S.I.R.I.S. — Protocollo Inquisitorio

Web app di produttività/task-tracking a tema dark/inquisitorio.
Nessun backend: tutti i dati restano salvati in locale sul dispositivo.

## Come usarla
Scarica lo ZIP del repo, apri `index.html` nel browser.

## Funzionalità
- Lista doveri giornalieri con streak e sistema di livelli
- Boss battle settimanale
- Backup/restore via JSON o codice
